## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Project Milestones

### Milestone 1: User Management (Weeks 1-2)
**Focus:** Establishing secure user authentication and role-based access control.

- **Key Achievements:**
  - Developed robust user registration and authentication systems.
  - Implemented Role-Based Access Control (RBAC) to distinguish between Volunteers and NGOs.
  - Created user profile management features, allowing users to view and edit their details.
  - Integrated a secure backend database for reliable user information storage.

- **Outputs:**
  - Fully functional login and registration interfaces.
  - Customized dashboards tailored to specific user roles.

### Milestone 2: Opportunity Management (Weeks 3-4)
**Focus:** Enabling NGOs to manage and broadcast volunteering opportunities.

- **Key Achievements:**
  - Enabled NGO administrators to create, update, and delete volunteering opportunities.
  - Designed comprehensive opportunity records that include essential details like required skills, duration, and descriptions.
  - Implemented a scalable database schema to manage opportunity data efficiently.

- **Outputs:**
  - User-friendly forms for opportunity creation and editing.
